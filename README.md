# 🤖 Telegram bots [Salebot]
## В данном репозитории описываются коммерческие low-code решения в виде телеграм-ботов на базе онлайн-конструктора Salebot.

---

## ❓ Почему именно Salebot

Онлайн-конструктор Salebot был выбран в качестве инструмента разработки чат-ботов по следующим причинам:
- простота и скорость разработки;
- отсутствие требований владения Python / JS;
- относительная дешевизна для бизнеса для небольших и краткосрочных проектов;
- экономия человеческого ресурса;
- готовый веб-интерфейс для администрирования;
- большое сообщество разработчиков Salebot позволяет быстро найти специалиста для найма и поддержки уже существующих проектов.
